# LUMINARA | Pranav P R — Art Gallery

Luminara is a high-end personal art portfolio designed to showcase a curated collection of color pencil portraits. It offers an immersive, cinematic experience through modern web aesthetics and fluid interactions.

## Key Features

- **Responsive Art Gallery**: A sophisticated grid layout featuring artworks in signature multilayered blue frames, optimized for mobile and desktop.
- **Dynamic User Experience**: Smooth Single Page Application (SPA) transitions and a unique canvas-based interactive background.
- **Cinematic Aesthetics**: A theme defined by deep navy tones, gold accents, and fluid reveal animations.
- **Direct Engagement**: Integrated contact automation for professional inquiries.

## Technical Overview

- **Frontend**: Semantic HTML5 and modern Vanilla CSS.
- **Logic**: Custom JavaScript (ES6+) for page orchestration and interactive elements.
- **Performance**: Lightweight and fast-loading with no external dependencies.
