# LUMINARA | Pranav P R - Art Gallery

Welcome to **LUMINARA**, a premium, high-end digital art gallery showcasing the exquisite color pencil portraits of Pranav P R. This project is built to deliver an immersive, cinematic experience for viewers, blending pixel-perfect realism with modern web aesthetics.

![LUMINARA Desktop Preview](/home/ppr/.gemini/antigravity/brain/6b60b923-497c-4a3a-8ac0-b0240f0d4dda/desktop_home_final_1771697304536.png)

## ✨ Features

- **🎨 High-End Gallery**: A curated collection of color pencil portraits, each enclosed in elegant, multilayered blue frames.
- **📱 Fully Responsive**: Optimized for every screen, from large desktop monitors to tablets and smartphones.
- **✨ Dynamic Animations**: Fluid, cinematic transitions and a unique canvas-based **Pixel Background** that reacts to interaction.
- **📧 Automated Contact Form**: Direct `mailto:` integration for instant professional inquiries.
- **⚡ Fast & Modern**: Built with pure HTML5, CSS3, and JavaScript for maximum performance and lightning-fast loading.

## 🛠️ Built With

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Vanilla CSS with modern Flexbox/Grid layouts and custom animations.
- **JavaScript**: Dynamic SPA-style page transitions and interactive canvas elements.
- **Vercel**: High-performance static hosting.

## 🚀 Deployment

This project is optimized for **Vercel**. 

If your code is in a subdirectory (e.g., `artgallery`), remember to set the **Root Directory** to `artgallery` in your project settings to ensure the site loads correctly at your main URL.
